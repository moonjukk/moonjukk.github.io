1. 개요

경기도 부천시 상동에 거주중인 5년차 java 웹 개발자.

성공회대학교 글로컬IT학과를 전공하고 소프트웨어공학과를 부전공하여 웹 개발을 공부하였고, 2021년 2월 SI 개발회사에 취직하여 개발자일을 시작하였다.
지금까지 4년간 4번의 프로젝트에서 백엔드엔드, 프론트엔드 분야를 가리지않고 시스템설계부터 개발, 운영까지 많은 경험을 하였다.

회사 경영상의 문제로 퇴사하여 기존사이트에서 8개월간 프리랜서로 추가로 근무.

2. 경력

2.1. 시드아이티

서울 금천구 가산디지털단지역 인근에 위치한 SI개발 회사로, 2021년 1월 입사하여 2월 첫 프로젝트로 LG U+ 차세대 개발 투입
1명~4명 개발자팀을 구성하여 파견하는 회사로, 모든 프로젝트는 서울에서 진행됨

3년간 총 4개의 프로젝트를 진행하였으며 회사의 경영상 문제로 퇴사

2.2. 프리랜서

2024년 3월 1일, 이전 회사를 경영상 문제로 급하게 퇴사하게되었으나 푸본현대생명 오토콜시스템 개발/운영 수행사인 Soluvis의 요청으로 8달간의 시스템 유지보수 및 고도화팀에 인수인계 작업 등을 요청받아 프리랜서 계약 후 수행 하였으며 2024년 10월 31일 게약만료

3.1. LG U+ 차세대 개발


마이플랫폼으로 기존 운영되던 관리자 프로그램을 웹 스퀘어를 사용하여 컨버팅하는 프로젝트로, 당시 빌링팀에서 UI/UX 화면개발을 수행

사용기술 : 자바, 웹스퀘어, 마이플랫폼, 오라클, git 등
수행기간 : 2021.02.01 ~ 2021.04.01

3.2. CJ대한통운 이커머스 eflexs 개발


CJ대한통운 이커머스를 운영하던 CJ올리브네트웍스팀에 개발 인력으로 투입되어 기존 마이플랫폼으로 운영되던 셀러 웹 프로그램을 JSP, jQuery, 스프링 등 을 활용해 컨버팅 작업 및 운영팀을 도와 유지보수 업무 수행
당일 배송시스템인 '오네'를 위해 판매물품을 센터에 입고시킨 판매자(관리자)가 맡긴 물품의 입고 부터 출고까지의 데이터를 관리할 수 있는 관리 웹 사이트를 개발

사용기술 : 자바, Spring, 마이플랫폼, JSP, jQuery, 오라클, SVN 등
수행기간 : 2021.04.01 ~ 2021.12.31

3.3. SERVEONE SSP 개발


서브원 SSP(서브원 서비스 플랫폼, MRO 산업재 B2B 웹 쇼핑몰) 개발 프로젝트로 프로젝트 설계부터 고객개발팀으로 투입, 운영자 사이드 웹(BO)은 넥사크로를 사용, 사용자 사이드 웹(FO)은 JSP, jQuery, 스프링 등 사용하여 개발.
고객의 가입부터 개인정보 관리, 휴면계정 관리, 고객별 할인율 관리 등 화면 개발및 오라클활용하여 쿼리 개발.
테스트 기간을 거쳐 운영팀에 약 6주간 인수인계 기간을 거쳐 프로젝트가 종료되어 철수.

사용기술 : 자바, Spring, JSP, jQuery, 오라클, 넥사크로, Git 등
수행기간 : 2021.01.01 ~ 2021.09.31

3.4. 푸본현대생명 오토콜시스템 개발/운영


푸본현대생명 오토콜 시스템 개발 및 운영 개발자로 IT기획운영팀에서 근무
- 대기중인 상담원과 고객을 자동 연결해주는 시스템인 해피콜 오토콜 시스템 운영을 메인으로 진행(실제 전화가 나가는 콜 시스템으로 까지 java 데몬 프로그램을 통해 전문으로 데이터를 통신하고 관리하며 오라클 DB로 데이터를 콜 시스템에 넘겨주는 것 까지의 작업)
- 통계 프로그램인 하이페리온을 사용하여 화면을 만들고 그 형태에 맞게 데이터를 채워주는 쿼리를 오라클 패키지를 활용해 개발 및 배치 프로그램인 데몬(jar파일) 운영 및 신규개발, 로그관리 등의 유지보수
- 자동아웃바운드 ARS 자바 데몬 시스템 개발, 전광판 웹 페이지 개선 작업 등의 신규 개발 건 동시 진행

사용기술 : 자바, Spring, JSP, 오라클, Git, 하이페리온 등
수행기간 : 2022.10.01 ~ 2024.10.31

4.1. 예술품 경매 사이트 제작

ArtAuction - 예술가를 위한 경매시스템 기반의 웹 오픈마켓


시연영상 : YouTube 바로가기
GitHub : GitHub 바로가기
PPT문서 : PPT 바로가기

개요

개인예술가들의 작품 공유 플랫폼의 부족
소규모 개인 예술가들이 작품을 쉽게 판매/공유 할 수 있는 오픈마켓 페이지의 개발요구
예술가들은 작품의 가치를 스스로 책정하고 경매를 통해 보상받는 방식

역할

5인의 팀원 중 팀장으로 수행하여 전체적인 개발 총괄
이미지 등 파일의 서버등록 및 관리, 경매 기능 시스템 , 웹 소캣 활용 등의 기능 구현 등

개발환경

언어 : Java, html, css, jQuery, JavaScript
데이터베이스 : Oracle(Spring) , MySQL(Node.js)
웹서버 : tomcat
프레임워크 : Spring, MyBatis, Node.js
스프링 라이브러리 : SpringSecurity, Lombok, Quartz, websocket 등

기능

▷ 개인예술가가 작품을 경매등록하고, 이용자들은 입찰하여 낙찰되면 개인간의 거래를 유도
▷ 보기 쉽게 분류가 가능한 상품리스트
▷ 스프링 Security를 사용하여 안전한 개인정보 관리
▷ 판매를 위한 경매 등록 시 이미지 파일을 ajax로 서버에 등록 및 DB에 기록하여 관리
▷ 스프링 Quartz라이브러리를 활용한 자동적 경매데이터 및 이미지파일 관리
▷ 웹소켓을 활용한 특정유저에게 메세지(알람) 전달기능
▷ Node.js를 사용해 별도의 소프트웨어 없이 고객과 상담사가 서로 채팅이 가능한 1:N 웹 채팅구현
▷ 관리자(상담사)의 회원 및 물품관리가 가능한 관리자페이지
